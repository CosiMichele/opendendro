# The dplPy User Manual

Welcome to the dplPy manual.

## Requirements

* Python v3.6+
* Pip

Recommended: 
* `Miniconda3` 
* `JupyterLab`

### Installation

```
pip install dplpy
```

```
conda install -c conda-forge dplpy
```

## Usage

From the command line interface (CLI), e.g., BASH, ZSH, or a Cygwin terminal 

#### help menu

Echos the help menu on the CLI 
```
python dplpy -h
```

#### `readme`

Opens this webpage

```
python dplpy readme
```
